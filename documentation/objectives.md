# Project Objectives

The primary objectives of this project are:

1. Calculate the overall customer churn rate.
2. Analyze customer demographics including age, gender, education, marital status, and income.
3. Identify customer segments with higher churn rates.
4. Evaluate credit card usage across different card categories.
5. Measure total transaction revenue and average customer spending.
6. Identify high-value customers based on transaction amount.
7. Analyze credit limits across different customer segments.
8. Build interactive dashboards for business users using Looker Studio.
9. Generate business insights that support customer retention strategies.
10. Demonstrate SQL analytics skills using aggregate functions, CASE statements, and window functions in BigQuery.
